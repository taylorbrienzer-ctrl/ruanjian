# 灵山 AI 数字人导览 uni-app 工程

这是移动 App 版工程骨架，用于交给 uni-app 开发继续实现。

## 页面

```text
pages/guide/index.vue     首页导览
pages/routes/index.vue    路线推荐
pages/spots/index.vue     景点讲解
pages/service/index.vue   行程服务
pages/profile/index.vue   我的
```

## 运行方式

推荐使用 HBuilderX：

1. 打开 `uni-app-lingshan` 目录。
2. 运行到浏览器、微信小程序或 App 模拟器。
3. 后续将页面内 mock 数据替换为真实接口。
